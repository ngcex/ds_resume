Invited talk at **OSINT Italia** (28 November 2024) on *Data-Driven Storytelling* — how data science tools and techniques can transform raw information into structured, actionable narratives.

The session walked through a practical pipeline spanning four areas:

1. **Open data** — sourcing and querying public datasets (municipal, regional, national)
2. **Web scraping** — automated extraction with Python and Selenium
3. **Social media monitoring** — Twitter/X analysis for tracking trends and public opinion over time
4. **Generative AI** — using LLMs for automated categorisation and summarisation of large document corpora

Each section was backed by live Jupyter notebooks, available in the [conference repository](https://github.com/ngcex/conference/tree/main/Osintitalia_28_11_2024), together with the full [slide deck (PDF)](https://github.com/ngcex/conference/blob/main/Osintitalia_28_11_2024/presentazione_osintitalia_CS_28_11_2024.pdf).

OSINT Italia is a community of practitioners at the intersection of open-source intelligence, investigative journalism, and data science.
